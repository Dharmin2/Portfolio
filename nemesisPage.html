Nemesis
Inventory Management

This is an inventory management system that was created for Nemesis Video Game Lounge.
This is a web application that helps keep track of their inventory and provides a few other features.

Programming languages used

  • Php
  • MySQL
  • HTML
  • CSS

Features of the Web Application

  • Keeping track of various types of items such as items they have in stock and items they have discarded
  • A system where admins can add items they are considering adding to the menu. These items can be voted for by customers
  • Customers can communicate with owners through the web application by sending them emails
