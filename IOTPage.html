IOT
Dashboard

This IOT dashboard incorperates various sensors to capture live data.
This data is used by the application to make decisions on its own without user input.

Technologies and programming languages used

  • Python
  • C++
  • Dht11
  • Photoresistor
  • NodeMCU ESP-8266
  • Step Motor
  • RFID

Features of the dashboard

  • Capturing temperature and humidity with the DHT11 sensor and displaying live data on the dashboard
  • Capturing light intensity with the photoresistor and displaying live data on the dashboard
  • Using Publish and Subcribe communication protocol to send data which is captured on the NodeMCU ESP-8266 to the Rasberry Pi
  • Automically sending emails to a user if the temperature exceeds a threshold. This email asks the user if they want to turn on the fan
  • Using RFID to personalize temperature, humidity and light thresholds for each user
